# 02. データ分析スキル (Analysis)

`01-data-skill.md` にて取得された最新のJSONデータを読み込み、以下のロジックに従って相場の解釈（定性・定量分析）を行ってください。

## 分析ロジック要件
1. **売買フロー底打ち検知 (`short_selling.json`)**:
   売買代金の膨張シグナル（S1：20日MA比+30%超）、および空売り比率ピークアウトシグナル（S2：直近ピークから相対で20%低下）を計算し、「Phase（下落 / 反発準備 / 反転上昇）」の状況を判定する。
2. **米系投資銀行のオプション手口分析 (`teguchi.json`, `option_history.json`)**:
   ゴールドマンサックス、J.P.モルガン、モルガン・スタンレーなど「米系投資銀行」のオプション・先物手口に絞って建玉の偏りを解析し、彼らが想定している相場の上値抵抗線（レジスタンス）と下値支持線（サポート）、および仕掛けの意図を推測する。
3. **セクター分析 (`etf_data.json` 等)**:
   必ず `etf_data_manager.py` が生成した日次データを用いて主要セクターETFの絶対値パフォーマンスを比較し、「現在日本市場で最も買われているセクター」と「最も売られている足枷のセクター」を明確に判定する。
4. **個別株ランキング (`etf_intraday.json`)**:
   必ず `fetch_intraday.py` で生成された JSONデータを用い、TOPIX100全銘柄の直近パフォーマンスを計算し、**「ランキング上位20銘柄」** と **「下位20銘柄」** を正確に抽出する。
5. **各種インデックスの評価 (`etf_intraday.json` 等)**:
   US先物指数（NQmain, ESmain, YMmain等）および、日本主要指数（日経平均 ^N225, TOPIX 1306.T, 半導体ETF 2644.T）の騰落結果（%）を取得・評価する。
   
**終了条件**:
上記の解析から「相場の現状を示すファクトと考察」が全て揃った段階で、次のレポート出力ステップ（03）へ進んでください。
